    # 防抖和节流

    # 为什么需要防抖、节流

    防抖、节流都是防止事件频繁触发的优化手段。但是它们处理的原理不通，导致它们的触发时机不一样。

    # 一、防抖

    # 1.什么是防抖

    在事件被触发n秒后再执行回调，如果在这n秒内又被触发，则重新计时。

    例如：一部电梯频繁有人进来，则不会关门，直到一段时间没有人进来，才会关门

    # 2.如何实现防抖

    function debounce(fn, delay = 100) {
  let timer;

  return function () {
    if (timer) clearTimeout(timer);
    timer = setTimeout(() => {
      fn.apply(this, arguments);
      timer = null;
    }, delay);
  };
}

    # 3.使用场景

    • Input输入框搜索功能
    • 窗口resize、scroll

    # 二、节流

    # 1.什么是节流

    规定在一个单位时间内，只能触发一次函数。如果这个单位时间内触发多次函数，只有一次生效。

    例如：射击游戏的射速，就算一直按着鼠标射击，在规定射速内，只能射出一发子弹

    # 2.如何实现节流

    function throttle(fn, delay = 500) {
  let timer = Date.now();
  
  return function () {
    if (Date.now() - timer > delay) {
      fn.apply(this, arguments);
      timer = Date.now();
    }
  };
}
    # 3.使用场景
    • 鼠标多次点击，单位时间内只触发一次
