    # HTML5语义化与新特性

    # 1.什么是HTML语义化

    语义化标签旨在，让标签具有含义。

    # 语义化的优点有：

    1.适合搜索引擎检索

    2.适合阅读，利于团队维护

    # 用法就是：在合适的地方使用合适语义化标签。

    # 例如遇到标题就用H1~H6，遇到段落就用P，边栏就用aside，导航就用nav等

    # 2.为什么要使用语义化标签

    • 在没有css样式的情况下，页面整体也会呈现很好的结构效果
    • 代码结构清晰，方便团队开发与维护
    • 更有利于搜索引擎优化

    # 3.HTML5新特性有哪些

    • 语义化标签
    • 音视频处理
    • canvas / webGL
    • history API
    • requestAnimationFrame
    • 地理位置
    • webSocket

    # 4.HTML5有什么新标签

    布局相关:header、main、footer、nav

    多媒体：video、audio、svg、canvas
